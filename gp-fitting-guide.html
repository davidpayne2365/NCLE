Gas Permeable Contact Lens Fitting Guide

Why SAM/FAP Matters

Let's start with WHY we use SAM/FAP. Sometimes we need to tighten or loosen a lens by adjusting the base curve. When we do this with a GP lens, it affects the tear lens (lacrimal lens) between the back surface of the contact lens and the cornea.

Visual Explanation

Let's visualize this:

Fitting on K

  • When you fit "on K," you're matching the base curve to the flattest curve of the cornea
  • The lens mirrors the corneal curvature
  • No power adjustment needed

Steeper than K

  • When the base curve is steeper than K, it creates a gap
  • This gap fills with tears, forming a plus-powered tear lens
  • Plus power converges light
  • To neutralize this unwanted plus power, we need to add minus power to the prescription
  • Hence: Steeper Add Minus (SAM)

Flatter than K

  • When the base curve is flatter than K, it creates a different tear lens shape
  • This forms a minus-powered tear lens
  • Minus power diverges light
  • To neutralize this unwanted minus power, we need to add plus power to the prescription
  • Hence: Flatter Add Plus (FAP)

Practical Example

Let's work through an example:

Example Data:

  • Refraction: -4.50 -0.75 x 180
  • K readings: 43.00 @ 180 / 43.75 @ 90

  1. Analyze K Readings

    • Flatter K is 43.00
    • Steeper K is 43.75
    • Difference is 0.75D (matches the astigmatism in refraction)
    • Steeper meridian is at 90° = With-the-rule astigmatism
      • (Within 30° of 90° = with-the-rule)
      • (Within 30° of 180° = against-the-rule)
      • (In between = oblique astigmatism)

  2. Determine Astigmatism Type

    • Corneal astigmatism (0.75D) matches total astigmatism (0.75D)
    • This means it's all corneal astigmatism (no lenticular component)
    • With-the-rule astigmatism can be corrected with GP lens up to 3D

  3. Ensure Minus Cylinder Form

    • Our prescription is already in minus cylinder form
    • If it were in plus cylinder, we'd need to transpose it
    • We always work in minus cylinder form for contact lenses

  4. Compensate for Vertex Distance

    • Check if power is ≥ 4D (yes, -4.50D)
    • Contact lens prescriptions become more plus/less minus than spectacle Rx
    • After compensation: -4.25D sphere
    • We can disregard cylinder and axis since GP will correct the astigmatism

  5. Apply SAM/FAP for Final Power

    • If fitting on K: Final power would be -4.25 sphere
    • If fitting 0.50D flatter than K:
      • Apply FAP (Flatter Add Plus)
      • -4.25 + 0.50 = -3.75 sphere
    • If fitting 0.50D steeper than K:
      • Apply SAM (Steeper Add Minus)
      • -4.25 - 0.50 = -4.75 sphere

Order of Operations for GP Fitting

  1. Convert to minus cylinder form
  2. Compensate for vertex distance
  3. Determine lens type based on astigmatism analysis
  4. Apply SAM/FAP adjustment if needed

Quick Reference

  • SAM: Steeper Add Minus
  • FAP: Flatter Add Plus
  • With-the-rule: Steeper in vertical meridian (within 30° of 90°)
  • Against-the-rule: Steeper in horizontal meridian (within 30° of 180°)
  • GP lenses can correct up to 3D of with-the-rule astigmatism
